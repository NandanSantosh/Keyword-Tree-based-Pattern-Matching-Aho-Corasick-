# Keyword-Tree-based-Pattern-Matching-Aho-Corasick-
• Implemented the Aho–Corasick algorithm using Keyword Trees and failure links for efficient multi-pattern matching.
• Performed pattern search on large DNA sequences, locating all occurrences of multiple substrings in O(n+m+k) time.
• Compared execution time and pattern-comparison counts against naive shift-based string matching algorithms
